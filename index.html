List Tags

There are three(3) main types of list Tags: Unorderd list, Ordered list, and Descriptive list

Unorderd List

This is used when order does not matter. Example:
A list of fruit

  • Apple
  • banana
  • Grape

Ordered list

This is used when order matters. Example:
Things to do today

  1. Apple
  2. banana
  3. Grape

NB: The 'type' attribute can be used to change the numbering system:

  • type="1": Numeric (default) — 1, 2, 3
  • type="A": Uppercase letters — A, B, C
  • type="a": Lowercase letters — a, b, c
  • type="I": Uppercase Roman numerals — I, II, III
  • type="i": Lowercase Roman numerals — i, ii, iii

Descriptive list

This is used for terms and descriptions (often forgotten, but powerful).Common use cases: Definitions, FAQs, Glossaries, etc. Example:
List of frontend technologies

HTML
Markup language for structuring web content
CSS
Styles and layout for web pages
NB: You can have a list in another list (Nested lists). Nested lists are used for Dropdown menus, Categories, Site maps, etc.
